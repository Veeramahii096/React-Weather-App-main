# Weather App
It is a simple weather app built using open Weather Api and React Js

  # How to Run
  - Download the zip File
  - Unzip the downloaded file
  -  Open in Vs Code
  -  Run " npm install " in terminal
  -  Run " npm run dev " in terminal
  -  Open the link
